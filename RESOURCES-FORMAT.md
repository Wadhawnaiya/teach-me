# RESOURCES.md Format

`RESOURCES.md` is the single curated list of high-trust sources for this topic. All explanations, examples, and claims in lessons should primarily draw from here (or sources added here). Never invent authority.

## Structure

```md
# {Topic} Resources

## Knowledge

- [Book / Course / Paper / Video: Title — Creator](url)
  One-line description of what it covers exceptionally well. Use for: specific sub-topic or depth.
- [NPTEL / SWAYAM / NCERT / Official: Title](url)
  ...

## Wisdom (Communities & Practitioners)

- [r/Indian_Academia or specific subreddit / Discord / Telegram / Local group](url)
  Signal level and why it is high-trust. Use for: real questions, feedback, accountability.
- Local: {specific class, mentor, study circle}
  ...

## Gaps (optional but valuable)

- Area X currently lacks a strong, accessible source we have found. Priority for future discovery.
```

## Rules (Strict)

- **High-trust only**. Primary sources, recognized experts, peer-reviewed or exceptionally well-vetted educational work, strongly moderated communities. Marketing disguised as education is excluded.
- **Annotate every entry**. "Bare links die." One sentence on coverage + when to reach for it.
- **Group by Knowledge / Wisdom**. Matches the philosophy. A source can live in one or both.
- **Surface gaps explicitly**. Honest gaps drive better future research.
- **Prune ruthlessly**. Wrong, shallow, or off-mission sources are removed, not archived.
- **Record preferences**. If the user has said "no communities for me", note it here so we stop suggesting them.
- **India priority when relevant**. When the topic has excellent Indian sources (NPTEL, DIKSHA, great Indian authors/educators, government material, exam-specific prep), list them prominently and first for that context. Still include the global best where they add unique value.
- **Update continuously**. As you and the user discover what actually moves the needle, add and annotate.

This file is living documentation of trust.
